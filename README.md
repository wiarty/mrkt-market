# FaraxShop

Telegram-бот для продажи Telegram Stars с приёмом оплаты в **RUB** и **USDT**
через [@CryptoBot](https://t.me/CryptoBot).

В репозитории два независимых проекта:

- `bot_system.py` — отдельный userbot-сервис (легаси, не используется FaraxShop).
- `farax_shop/` — пакет основного бота FaraxShop.

## Возможности

- Главное меню с кнопками **Купить звёзды**, **Профиль**, **Мои заказы**,
  **Канал**, **Поддержка**.
- Покупка звёзд:
  - выбор количества из пресетов или произвольное число;
  - выбор валюты — RUB или USDT;
  - выставление счёта в CryptoBot, кнопка «Оплатить» → CryptoBot;
  - автоматическая проверка оплаты (фоновый поллер) + ручная кнопка
    «Я оплатил — проверить»;
  - после оплаты пользователь получает уведомление, админ — заказ
    с кнопкой «Отметить выданным».
- Профиль с суммарной статистикой (потрачено, куплено).
- Админ-панель `/admin`:
  - **Цены**: цена за 1 звезду в RUB и USDT;
  - **Минимум звёзд** на заказ;
  - **Поддержка**: username для кнопки «Поддержка»;
  - **Канал**: ссылка на канал;
  - **Рассылка** любого сообщения (текст/медиа) всем пользователям;
  - **Статистика** заказов, пользователей и выручки;
  - **Заказы**: список pending-заказов и управление ими.
- Все сообщения и кнопки используют **премиум-эмодзи** через
  `<tg-emoji emoji-id="...">…</tg-emoji>` и `icon_custom_emoji_id`.

## Запуск

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

cp .env.example .env
# Заполни BOT_TOKEN, ADMIN_IDS и CRYPTOBOT_TOKEN

python -m farax_shop.main
```

## Переменные окружения

| Переменная | Описание |
|---|---|
| `BOT_TOKEN` | Токен бота от @BotFather. |
| `ADMIN_IDS` | Список Telegram ID админов через запятую. |
| `SHOP_NAME` | Имя магазина (по умолчанию `FaraxShop`). |
| `DB_PATH` | Путь к SQLite-файлу (по умолчанию `farax_shop.db`). |
| `CRYPTOBOT_TOKEN` | Токен от @CryptoBot (`Crypto Pay → Create App`). |
| `CRYPTOBOT_TESTNET` | `1` для использования testnet (@CryptoTestnetBot). |
| `INVOICE_POLL_INTERVAL` | Интервал поллинга счетов в секундах (≥ 5). |

## Структура

```
farax_shop/
  __init__.py
  main.py              # точка входа
  config.py            # загрузка .env
  database.py          # SQLite (aiosqlite)
  cryptobot.py         # клиент Crypto Pay API
  emojis.py            # справочник премиум-эмодзи
  texts.py             # шаблоны сообщений
  keyboards.py         # клавиатуры (inline / reply)
  states.py            # FSM-состояния
  poller.py            # фоновая проверка оплат
  handlers/
    __init__.py
    start.py           # /start, меню, профиль, мои заказы
    buy.py             # покупка звёзд
    payment.py         # подтверждение оплаты, отмена
    admin.py           # админ-панель и настройки
    broadcast.py       # рассылка
```

## Доставка звёзд

Бот фиксирует факт оплаты и присылает админу карточку заказа с кнопкой
«Отметить выданным». Сама выдача звёзд (через Fragment / @PremiumBot и т.п.)
выполняется админом вручную — это самый универсальный сценарий и не требует
интеграции с Fragment-кошельком в боте.

## Премиум-эмодзи

Идентификаторы эмодзи собраны в `farax_shop/emojis.py`. Для текста
используется `tg(emoji_id, "fallback")` или короткая обёртка `e(emoji_id)`,
для кнопок — `icon_custom_emoji_id` без unicode-эмодзи в самом `text`.
